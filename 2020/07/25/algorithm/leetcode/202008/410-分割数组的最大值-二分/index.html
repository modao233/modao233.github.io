leetcode410 分割数组的最大值，涉及算法：二分查找、动态规划

题目描述

给定一个非负整数数组和一个整数 m，你需要将这个数组分成 m 个非空的连续子数组。设计一个算法使得这 m 个子数组各自和的最大值最小。

注意:
数组长度 n 满足以下条件:

  • 1 ≤ n ≤ 1000
  • 1 ≤ m ≤ min(50, n)

示例:

解题思路

二分查找

由题意可知：子数组的最大值是有范围的，即在区间 [max(nums),sum(nums)] 之中。
令 l=max(nums)，h=sum(nums)，mid=(l+h)/2

使用二分法去逐渐逼近， 令 mid = (left + right) / 2 为 (每一种划分情况)子数组各自和的最大值 的最小值， 那么所有的子数组和小于或等于 mid

计算数组和最大值不大于mid对应的子数组个数 cnt(这个是关键！)

  • 如果 cnt>m，说明划分的子数组多了，即无论怎么切都不能所有的子数组和 <= mid，即我们找到的 mid 偏小，故 l=mid+1；
  • 否则，说明划分的子数组少了，即 mid 偏大(或者正好就是目标值)，故 h=mid。

动态规划

dp[i][j]：将数组的前i个数分割成j段所能得到的最大连续子数组和的最小值
