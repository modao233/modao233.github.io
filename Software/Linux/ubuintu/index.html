    记录下安装Ubuntu的一些基本配置

    安装完后的基本配置

    1. 换国内源

      • 先备份一份软件源配置：sudo cp /etc/apt/sources.list /etc/apt/sources_bak.list

      • 修改源文件： sudo vi /etc/apt/source.list

        deb https://mirrors.ustc.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse

        deb-src https://mirrors.ustc.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse

        ```

    2. 打开“软件和更新”，把五个选项都勾选，然后下拉框里点其他站点选择USTC的源

    3. 更新索引：sudo apt-get update

    4. 更新软件：sudo apt-get upgrade

    5. 打开语言支持，如果选择安装时简体中文可能没下载完整，打开后有弹框选择下载

    6. 下载完后点应用到整个系统，注销再登录就可以了
