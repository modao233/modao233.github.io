频率、时间的换算

Hz 是频率的单位。频率是指电脉冲，交流电波形，电磁波，声波和机械的振动周期循环时，1秒钟重复的次数。

Hz是个很小的单位，通常在其前面加上k（千），M（百万），G（十亿），T（万亿）等数量级单位。

1 Hz = 1 次/s

1 kHz = 1000 Hz

1 MHz = 1000 kHz

1 GHz = 1000 MHz

1 THz = 1000 GHz

秒的换算

ms(毫秒)、μs(微秒)、ns(纳秒)、ps(皮秒)

1s = 1000 ms

1ms = 1000μs

1μs = 1000ns

1ns = 1000ps
