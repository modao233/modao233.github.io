OpenCV C++项目环境配置

2020-07-07

windows10系统，VS2019

配置环境变量

系统环境变量，path变量：

 C:\opencv\build
 C:\opencv\build\x64\vc14\bin

配置项目属性

  1. VC++目录 包含目录：

    C:\opencv\build\include
C:\opencv\build\include\opencv
C:\opencv\build\include\opencv2

  2. VC++目录 库目录：

    C:\opencv\build\x64\vc14\lib

  3. 链接器 输入 附加依赖项【对应opencv版本】：

    opencv_world345.lib
opencv_world345d.lib
