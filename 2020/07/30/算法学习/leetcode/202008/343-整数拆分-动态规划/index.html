leetcode 343 整数拆分，涉及算法：动态规划（含优化）、数学归纳

题目描述

给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

说明: 你可以假设 n 不小于 2 且不大于 58。

解题思路

动态规划

  • 创建数组 dp，其中 dp[i] 表示将正整数 i 拆分成至少两个正整数的和之后，这些正整数的最大乘积
  • 对于 数字 n，可以拆分为 i 和 n-i，i 的范围是 1 到 n-1，遍历此范围
    • 对n - i，可以选择拆分和不拆分，取最大值
    • 将每种拆分状态的最大值进行比较，取其最大的存入数组dp[n]

复杂度分析

  • 时间复杂度：$O(n^2)$，其中 n 是给定的正整数。对于从 2 到 n 的每一个整数都要计算对应的 dp 值，计算一个整数对应的 dp 值需要 O(n) 的时间复杂度，因此总时间复杂度是 $O(n^2)$。

  • 空间复杂度：O(n)，其中 n 是给定的正整数。创建一个数组 dp，其长度为 n+1。
