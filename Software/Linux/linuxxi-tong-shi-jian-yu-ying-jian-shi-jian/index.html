Linux的系统时间和硬件时间

2020-02-28
:: tags:  #Software  #Linux

最近在虚拟机里更新软件时遇到了一个问题导致无法获取更新，经排查后发现是因为我当时是暂停虚拟机而不是关闭，导致系统时间暂停，系统时间和软件镜像源源不一致，导致无法获取更新。

查看系统时间：

date

设置系统时间：

date -s + 时间

设置系统时间和日期：

date -s "2018-08-08 10:00:00"

查看硬件时间：

hwclock 或 clock

将系统时间写入到硬件时间：

hwclock -w

将硬件时间写入到系统时间：

hwclock -s

时间同步，Linux系统安装ntpdate(时间同步命令)

ntpdate www.pool.net.org 
#这个网站是互联网中标准时间服务器
