Navicat v15注册时报错“rsa-public-key-not-find”的解决办法

安装包版本：navicat150_premium_cs_x64

注册机版本：Navicat_Keygen_Patch_v5.6_By_DFoX

遇到的问题：rsa-public-key-not-find

无效的办法：删注册表

有效的办法：卸载干净后重新安装，安装完不要开软件，直接开注册机打补丁，接着直接注册

关键点：补丁只能打一次，失败后必须重新卸载安装
