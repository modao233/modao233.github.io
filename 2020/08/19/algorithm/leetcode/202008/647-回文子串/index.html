leetcode 647 回文子串，涉及算法：中心拓展、动态规划、manache

题目描述

给定一个字符串，你的任务是计算这个字符串中有多少个回文子串。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。

示例：

提示：

  • 输入的字符串长度不会超过 1000 。

解题思路

中心拓展

回文串是中心对称的：

  • 字符个数为奇数：中心为一个字符
  • 字符个数为偶数：中心为两个字符

寻找回文子串最暴力的解法是对字符串的每个子串判断是否是回文串，则时间复杂度高达$O(n^3)$

为了减少时间复杂度，将遍历寻找回文子串改成遍历寻找回文串中心：

字符串长度为N的字符串，遍历的中心数为：

  • 假设回文子串为奇数字符，中心只有一个字符，遍历数为 N
  • 假设回文子串为偶数字符，中心只有一个字符，遍历数为 N-1

合计遍历 2N-1 次。假设每个回文字符串都有两个中心字符，奇数个的两个字符为同一个，则中心字符有：

  • $l = i \div 2$
  • $r = l + (i \quad mod \quad2)$

复杂度分析

  • 时间复杂度：$O(n^2)$
  • 空间复杂度：$O(1)$

动态规划

用 dp 数组来进行动态规划, dp[i][len] 表示从 i 开始的长度为 len 的子串是否是回文子串, 那么 dp[i][len] = dp[i + 1][len - 2] && s[i] == s[i + len - 1], 以此来动态规划即可, 其实和中心扩散本质是一样的, 都是从小区间向大区间扩散。
