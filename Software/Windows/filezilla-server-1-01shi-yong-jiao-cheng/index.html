Windows系统-FTP服务器软件

2022-01-03
:: tags:  #Software  #Windows
  1. 下载安装包，安装
  2. 设置相应信息
  3. 创建组，指定组目录，创建用户，指定用户目录
  4. 允许防火墙通过 filezilla server
  5. 新版服务端编码方式为UTF-8，使用win10的资源管理器拖文件时会编码不一致，中文文件出错

主要问题是：

  1. 版本与已有文档不一致，了解完后其实主要就是创建组和用户并指定相应目录及权限
  2. 搭建成功，本地可以访问，其它PC访问不到，原因是防火墙拦截住了，添加防火墙白名单即可
  3. win10资源管理器上传中文文件出错，原因是编码不一致，新版server程序默认是UTF-8，解决方法就是换UTF-8编码的客户端，如winscp
