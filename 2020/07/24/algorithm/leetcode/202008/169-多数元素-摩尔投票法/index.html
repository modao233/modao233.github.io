leetcode169 多数元素，涉及算法：摩尔投票法

算法描述

摩尔投票法（Boyer–Moore majority vote algorithm），也被称作多数投票法，算法解决的问题是：如何在任意多的候选人中（选票无序），选出获得票数最多的那个。

算法可以分为两个阶段：

  1. 对抗阶段：分属两个候选人的票数进行两两对抗抵消
  2. 计数阶段：计算对抗结果中最后留下的候选人票数是否有效

优点：时间复杂度为 O(N)，空间复杂度为 O(1)

LeetCode 169. 多数元素

给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。
