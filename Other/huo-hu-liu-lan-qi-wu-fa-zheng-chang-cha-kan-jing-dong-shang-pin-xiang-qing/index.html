火狐浏览器无法正常查看京东商品详情

2022-02-19
:: tags:  #Other

在使用火狐浏览狗东时，火狐优先使用ipv6地址，而京东物品详情页面对ipv6支持不好（京东主页支持ipv6），导致详情页面无法打开。

火狐 地址栏输入 about:config 回车。
搜索 ipv6
右击高亮标出的 'network.dns.disableIPv6，将 false 改为 true

火狐就能正常浏览京东了。
